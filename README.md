# danielfortune.com

- **Framework**: [Next.js](https://nextjs.org/)
- **Deployment**: [Vercel](https://vercel.com)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)

## Running Locally

```bash
$ git clone https://github.com/aaimeraud/fortunedaniel.com

$ cd danielfortune.com

$ bun install

$ bun dev
```
